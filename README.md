dice
====
### 安装

```bash
$ git clone https://github.com/adleida/dice.git
$ cd dice
$ sudo python3 setup.py install
```

### 帮助系统:

```bash
$ dice -h
```
> *-p: * 指定dice运行时的port(默认值为: 6060)
  *-c: * 指定dice运行时的config文件(默认为: /usr/local/lib/python3.4/dist-packages/dice/res/dice.yaml)
  *-v: * 获取dice 当前的版本号

### 运行dice
```bash
$ dice -p 6001
```
